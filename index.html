Content

It all starts with content. No content, there is nothing to design and nothing to code

Design

Once you have your content, you need to think about how you are going to present it> What will it look like, what colours to use, how to engage with the audience and tell the story..

Coding

Once you have ordered the content and have visualised the display, the challenge is to write code to make it appear, and react to the user, in the browser.
